### Project Knight

A small 2D combat-focused game demo made purely with C++ and SDL2.

## Usage

```
cd src
./main
```

You may need to install the SDL2 and SDL2 image libraries.

## Controls

- W : Jump
- A : Move left
- S : Crouch
- D : Move right

- J : Attack
- K : Roll
- L : Heal

- S + J while in the air : Air attack
